SAP BP Profile Sync

An ETL pipeline for extracting Business Partner profiles from SAP and synchronizing them with a SQL Data Warehouse.

Project Overview

Designed and implemented a robust data pipeline to automate the extraction of business partner data from SAP systems and transform it for use in a centralized data warehouse.

The solution reduced manual data processing time by 85% and improved data accuracy for reporting and analytics.

Technical Details

Extraction: SAP BAPI and RFC calls

Transformation: Python, Pandas for data cleansing

Loading: SQL Server Integration Services (SSIS)

Storage: Microsoft SQL Data Warehouse

Scheduling: Apache Airflow

SAP BAPI Python Pandas SSIS MS SQL Airflow

Key Features

  • Automated daily data extraction from SAP
  • Data validation and cleansing processes
  • Incremental loading to optimize performance
  • Error handling and logging mechanisms
  • Data quality reporting
  • Monitoring dashboard
